## Overview
AgentWrite-Blog-AI is an AI agent system that automates the entire blog creation process. It can generate blog topics, create full drafts, proofread, and format outputs with references, saving content creators time and effort.

## Features
- **Topic Generation:** Suggests multiple blog topics based on user keywords.
- **Draft Writing:** Generates detailed, engaging blog posts using AI.
- **Proofreading & Editing:** Corrects grammar, improves readability, and ensures coherent structure.
- **References:** Appends relevant references automatically.
- **Markdown Export:** Saves ready-to-publish blog drafts in Markdown format.
